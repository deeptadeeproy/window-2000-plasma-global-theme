# 🌬️ Windy 2000 (Global Plasma Theme)

An authentic, modern global theme for KDE Plasma that blends the best aesthetic features of mid-2000s desktop operating systems. `Windy 2000` brings back the glassy, tactile, and paper-like elements of classic Windows and macOS interfaces while maintaining a clean workflow for modern Linux platforms.

---

## ✨ Features

*   **Mid-2000s Hybrid Aesthetic:** Merges the iconic glass borders of Windows Vista/7 Aero with the refined, paper-like brushed textures of early Mac OS X Panther/Tiger.
*   **Tactile Pane Styling:** Curved panel styles designed to look stunning in both docked layouts and modern floating configurations.
*   **Retro-Modern Icon Synergy:** Tailored to accent perfectly with high-fidelity, skeuomorphic, and dark-mode icon grids (like Tela-dark or classic Aqua/Luna remixes).
*   **Glass & Shadow Profiles:** Pre-configured KWin window decoration presets that render soft drop shadows and realistic frosted glass blurs.

---

## 🚀 Installation

### Method 1: The Fast Route (KDE Store)
1. Open **System Settings** -> **Colors & Themes** -> **Global Themes**.
2. Click **Get New Global Themes...** at the bottom right.
3. Search for `Windy 2000`, click **Install**, and apply.

### Method 2: Manual Git Installation
Clone the repository directly into your local user space directory:

```bash
mkdir -p ~/.local/share/plasma/desktoptheme/
git clone [https://github.com/yourusername/windy-2000.git](https://github.com/yourusername/windy-2000.git) ~/.local/share/plasma/desktoptheme/Windy2000
